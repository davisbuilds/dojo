# Font Pairings by Aesthetic Direction

Curated Google Fonts pairings. Each entry: display font (headings) + body font (text), with import URL.

---

## Brutalist / Raw

**Clash Display + IBM Plex Mono**
Heavy, unapologetic display type with monospace body. Feels industrial and confrontational.
```
@import url('https://fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@400;500;700&display=swap');
/* Clash Display: available via FontShare or self-host */
```

**Bebas Neue + JetBrains Mono**
All-caps condensed headings with developer-grade mono body. Raw, utilitarian.
```
@import url('https://fonts.googleapis.com/css2?family=Bebas+Neue&family=JetBrains+Mono:wght@400;500;700&display=swap');
```

**Anton + Source Code Pro**
Bold, compressed display with readable mono. Poster-like headings, terminal-style body.
```
@import url('https://fonts.googleapis.com/css2?family=Anton&family=Source+Code+Pro:wght@400;500;700&display=swap');
```
