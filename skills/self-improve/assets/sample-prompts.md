# Sample Prompts

## Should Trigger `self-improve`

- "Keep a portable log of the agent's recurring mistakes and workflow learnings."
- "Compact these agent learnings into a smaller summary before we reuse them."
- "We keep repeating this fix. Propose a durable improvement and extract a draft reusable skill."

## Should Not Trigger `self-improve`

- "Update AGENTS.md with non-obvious learnings from this session."
- "Create a handoff summary so a fresh instance can continue."
- "Help me create a brand-new skill from scratch."
