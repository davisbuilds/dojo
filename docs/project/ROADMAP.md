# Roadmap

This is a lightweight snapshot, not a release contract.

## Completed Highlights

- 37 curated skills spanning GitHub workflows, code review, design, platform integrations, and knowledge management.
- 5-hook quality pipeline: session-start catalog injection, pre/post-tool-use validation and manifest sync, stop-time git and structure checks.
- Skill packaging and distribution via `.skill` zip format.
- Command wrappers for deterministic slash-style entrypoints.
- Skill installer supporting Claude Code and Codex destinations.
- Machine-readable `skills.json` manifest with auto-regeneration.
- Polyglot validation scripts compatible with both `python` and `python3`.

## Planned / Open Areas

- Skill versioning and changelog tracking.
- Expanded test coverage for hook scripts and validation logic.
